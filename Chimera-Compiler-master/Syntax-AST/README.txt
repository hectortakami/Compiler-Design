Chimera compiler, version 0.3
===============================

This program is free software. You may redistribute it under the terms of
the GNU General Public License version 3 or later. See license.txt for 
details.    

Included in this release:

   * Lexical analysis
   * Syntactic analysis
   * Abstract Syntax Tree
    
To build, at the terminal type:

    make
   
To run, type:

    ./chimera.exe <file_name>
    
Where <file_name> is the name of a Chimera source file. You can try with
these files:

   * hello.chimera
   * palindrome.chimera
   * variables.chimera
   * lists.chimera
   * binary.chimera
   * factorial.chimera
